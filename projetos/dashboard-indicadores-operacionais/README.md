# Dashboard de Indicadores Operacionais

Projeto de criação de dashboard para monitoramento de indicadores operacionais, com foco em desempenho, falhas e eficiência.

## Objetivo
Acompanhar métricas operacionais e apoiar a tomada de decisão através da visualização de dados.

## Indicadores
- Quantidade de falhas
- Tempo médio de atendimento
- Percentual de atendimento no prazo

## Ferramentas
- Power BI
- Excel

## Etapas
- Coleta e organização dos dados
- Modelagem simples
- Criação de visualizações
- Análise de indicadores

## Insights
- Identificação de variações no número de falhas
- Monitoramento da eficiência operacional
- Avaliação do cumprimento de prazos
